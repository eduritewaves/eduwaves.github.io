1. ಮತ್ತೆ ಸೇರ ಬೇಕಿದೆ...

ನನ್ನ ಪ್ರೀತಿ ಪಯಣವೇಕೊ ಅವಳ ಹಿಂದೆ ಹೊರಟಿದೆ...
ಚೂರು ಸದ್ದು ಮಾಡದೇನೆ ಹೃದಯ ಹಾರ ಹೊರಟಿದೆ...
ಕಂಗಳೇಕೊ ಮರಳಿ ಅವಳ ದನಿಯ ಹುಡುಕಿ ಹೊರಟಿದೆ...
ಹೇಳಲೇಗೆ ಹೇಳು ಗೆಳತಿ ನಿನ್ನ ಸನಿಹ ಬೇಕಿದೆ...
ಮರಳಿ ಮರಳಿ ಮರೆತ ಕ್ಷಣವು ನಿನ್ನ ನೆನಪು ಮಾಡಿದೆ...
ಮತ್ತೆ ಸೇರ ಬೇಕಿದೆ... ಮತ್ತೆ ಸೇರ ಬೇಕಿದೆ...

Name : Prathviraj N. M.
Course : B.C.A.
Semester : 3rd

2. ಮರಳಿ ನಿನ್ನ ಸೇರಲೇನು...

ಸಾಲು ಸಾಲು ಕವಿತೆ ಸಾಲ ಸಾರಿ ಸಾರಿ ಹೇಳಲೇನು...
ಉಸಿರಿಗೊಮ್ಮೆ ನಿನ್ನ ಹೆಸರ ಹೇಳಿ ಸುಮ್ಮನಾಗಲೇನು...
ಮರಳಿ ನಿನ್ನ ಸನಿಹ ಬಯಸಿ ಮತ್ತೆ ಪ್ರೇಮಿಯಾಗಲೇನು...
ಪ್ರೀತಿಯೊಂದೇ ಧ್ಯೇಯ ಮಾಡಿ ಜಗವ ಗೆಲ್ಲ ಹೊರದಲೇನು...
ಮತ್ತೆ ಪ್ರೇಮಿಯಾಗಲೇನು ಮರಳಿ ನಿನ್ನ ಸೇರಲೇನು...

Name : Prathviraj N. M.
Course : B.C.A.
Semester : 3rd

4. ಹದಿನೆಂಟರ ಹರೆಯ

ಆಕರ್ಷಣೆಗಳತ್ತ ಆಗುವ ಮನದ ಸೆಳೆತ ಆದರೆ ಸುಮ್ಮನಾಗಿಸುವ ಕಟ್ಟುಪಾಡುಗಳ ಬಿಗಿ ಹಿಡಿತ
ಕಣ್ಣಿಗೆ ಕಂಡದನೆಲ್ಲ ಬೇಕೆನ್ನುವ ಹುಚ್ಚುಕೋಡಿ ಮನ ಮೈಮರೆಯದಿರಿ ಹರಸಿ ಬೇರೆಡೆಗೆ ಗಮನ
ಮಾಡುತ್ತಿರುತ್ತಾರೆ ನಮ್ಮನ್ನು ಅಕ್ಕಪಕ್ಕದವರ ಜೊತೆ ಹೋಲಿಕೆ ಆದರೂ ಇರಲಿ ಸಮಾಧಾನ ಜೀವಕ್ಕೆ
ಅನಗತ್ಯ ವಿಷಯಗಳತ್ತ ಹೆಚ್ಚುವ ಕುತೂಹಲ ಎಚ್ಚರ ತಪ್ಪಿ ಮುನ್ನುಗಿದರೆ ಕುಡಿದಂತಾಗುವುದು ಹಾಲಹಲ
ಕೈಬೀಸಿ ಕರೆಯುವ ಮಾಯಲೋಕ ಮಾಡಿಬಿಡುತ್ತದೆ ನಮ್ಮನ್ನು ಬಲಿಯ ಮಿಕ Soooo... be aware ಆಗದಿರಲಿ ಜೀವನ ನರಕ
ಒಂದೆಡೆ ನಿಧಾನವಾಗಿ ಹೆಚ್ಚುತ್ತಿರುವ ಜವಾಬ್ದಾರಿಗಳ
ಮತ್ತೊಂದೆಡೆ ನಡೆಯುತ್ತಿರುತ್ತದೆ ವಿದ್ಯೆ- ಉದ್ಯೋಗದ ಚಿಂತನೆ
ಈ ಚಕ್ರವ್ಯೂಹವ ಭೇದಿಸಿ ಬರುವ ಕಲೆ ಹೇಗೆ ಕಲಿಯುವುದೋ ನಾ ಕಾಣೆ...... ಸವಾಲುಗಳನ್ನು ಮೆಟ್ಟಿನಿಂತು ಆಗಿರಿ ಸ್ಫೂರ್ತಿಯ ಚೆಲುಮೆ.

Name : Harsha Ramani
Course : B.Com
Semester : 3rd

5. ನೀನು ಒಂದು ಪುಸ್ತಕವೇ...

ಅಸಂಬದ್ಧ ಪುಟಗಳನ್ನು ಹರಿದು ಹಾಕಿ ಒಳ್ಳೆಯ ವಿಚಾರಗಳನ್ನು ಕೊನೆಗೆ ಟಿಪ್ಪಣಿಯಲ್ಲಿ ಸೇರಿಸಬೇಕಾಗಬಹುದು.
ಕಳೆದುಹೋದ ಹಾಳೆಗಳ ಹುಡುಕುವ ಬದಲು ಬದುಕಿನ ಹಳಿಯನ್ನು ರೂಪಿಸುವಂಥ ವಿಚಾರಧಾರೆಗಳನ್ನು ಬರೆದುಕೊಳ್ಳಬೇಕಾಗಬಹುದು.
ನಿನ್ನ ಮುಖ ಪುಟದ ಚೆಂದದಿಂದಲ್ಲ ನಿನ್ನೊಳಗಿನ ವಿಷಯದ ಅಂದದಿಂದ ಆಕರ್ಷಿಸುವಂತಾಗು.
ಎಲ್ಲರೂ ನಿನ್ನ ಕೈಗತ್ತಿಕೊಂಡು ಓದಿ ಪುಟಗಳು ಸವಿದು ನಿನ್ನ ಜೀವನವೂ ಸಾರ್ಥಕತೆಯ ಪೂರ್ಣವಿರಾಮ ಪಡೆಯುವಂತಾಗಲಿ.

Name : Harsha Ramani
Course : B.Com
Semester : 3rd

6. ಮನುಜ...

ಎಲ್ಲು ತಲುಪದ ದೋಣಿಯೇರಿ ಮೆರೆವನೋ ಈ ಹುಚ್ಚ ಮನುಜ
ಎಲ್ಲೆ ಇಲ್ಲದ ಬಾನನೇರುವ ಮೂಢನೋ ಈ ಹುಚ್ಚ ಮನುಜ
ಯಾರು ಕಾಣದ ಸತ್ಯದೆಡೆಗೆ ಎಂದು ಸಾಗದ ದಾರಿ ಕಡೆಗೆ
ಅಂತ್ಯವಿಲ್ಲದ ಕಾಲದೆಡೆಗೆ ಕೊನೆಯೆ ಇಲ್ಲದ ಸೋಲಿನೆಡೆಗೆ
ಪಯಣ ನಡೆಸಿದ ಅಲ್ಪ ಮನುಜ...

ಕಷ್ಟ ಬರಲು ಕರುಣೆ ಮರೆತ
ನಷ್ಟ ಬರಲು ರುಧಿರ ಮರೆತ
ಮನದ ಮಾತಿಗೆ ಕಿವಿಯ ಕೊಡದೆ
ಕರುಳ ಬಳ್ಳಿಯ ಋಣವ ಮರೆತ

ನಗಲು ಬೇಡ ನೂರು ನಾಚಿಕೆ
ನಕ್ಕಿ ಹೊರಬಾ ನೋವಿನಾಚೆಗೆ
ನಗಲು ಕಲಿತರೆ ಎಲ್ಲ ನೋವಿಗೆ
ಕಷ್ಟವೆಂಬುದು ಸುಖದ ಹಾಸಿಗೆ

Name : Kiran Udupa G. V.
Course : B.Com
Semester : 3rd

7. ಕಳೆಯ ಬೇಡ ಹೊತ್ತು...

ಗೂಡ ಬಿಟ್ಟು ಗುಬ್ಬಿಯೊಂದು ಹಾರುತಿಹುದು ಬಾನಲಿ
ಬಾನ ಎಲ್ಲೆ ದಾಟಿ ಮೆರೆವ ಜ್ವಾಲೆ ಇತ್ತು ಕಣ್ಣಲಿ
ಬಂಧು ಬಳಗ ಬಿಟ್ಟು ಬಂದ ನೋವು ತನ್ನ ಮನದಲಿ
ಹಾದಿ ತಪ್ಪೋ ಭಯವು ಅದಕೆ ಜಗವ ನೋಡೋ ಬರದಲಿ

ಮನೆಯ ನೆನೆದು ಮನಸ ತೊಳೆದು
ಮನಮುರಿಯುವ ಕೆಲಸ ಹಿಡಿದು
ಅಳುತ ನಗುವ ಭಾಗ್ಯ ಪಡೆದು
ತನಗಿರುವ ಸುಖವ ತೊರೆದು
ಗೂಡ ಬಿಟ್ಟು ಗುಬ್ಬಿಯೊಂದು ಹಾರುತಿಹುದು ಬಾನಲಿ

ನಾಳೆ ಒಂದು ಕಲ್ಪನೆ ಇಂದು ನಿನ್ನ ಸ್ವತ್ತು
ನೆನ್ನೆ ಒಂದು ಯೋಚನೆ ಕಳೆಯ ಬೇಡ ಹೊತ್ತು...

Name : Kiran Udupa G. V.
Course : B.Com
Semester : 3rd
